# Sales_report_analysis
# Project Overview
This project focuses on analyzing Vrinda Store sales data using Microsoft Excel to identify customer purchasing behavior, sales trends, top-performing states, and profitable sales channels. The analysis helps derive actionable business insights and supports data-driven decision-making.

# Objectives
Analyze overall sales performance Identify top-performing customer segments Understand state-wise sales contribution Analyze sales channels and order trends Generate business insights and recommendations

# Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Creation
- Data Cleaning
- Conditional Formatting

# Dataset Information
The dataset contains:
- Customer demographics
- Order details
- Sales amount
- State-wise sales
- Gender and age analysis
- Sales channel information

# Key Analysis Performed
- Sales Analysis
- Monthly sales trend analysis
- Order quantity analysis
- Customer Analysis
- Gender-wise sales comparison
- Age group contribution analysis
- Regional Analysis
- Top-performing states identification
- Channel Analysis
- Sales contribution by Amazon, Flipkart, Myntra, etc.

# Dashboard Features
- Interactive Excel Dashboard
- Dynamic Pivot Charts
- Filters using Slicers
- Sales performance visualization
- Customer segmentation insights

# Key Insights
1) Women customers contribute the highest share of sales (~65%)
2) Customers aged 30–49 years are the major contributors
3) Maharashtra, Karnataka, and Uttar Pradesh are the top-performing states
4) Amazon, Flipkart, and Myntra are the leading sales channels

# Final Conclusion
The analysis indicates that women customers aged 30–49 years contribute significantly to overall sales. Targeted marketing campaigns, personalized offers, and promotional advertisements through Amazon, Flipkart, and Myntra in high-performing states can improve customer engagement and increase sales performance.

# Business Recommendations
- Focus marketing campaigns on women customers
- Increase promotions in top-performing states
- Strengthen online sales channels
- Provide seasonal offers and discounts
- Improve customer retention strategies
